Project title:

To-do list:

Overview:

A user-friendly To-Do List application designed to help users manage their tasks efficiently, ensuring better productivity and organization.

Objective:

To develop an intuitive application where users can easily add, edit, delete, and mark tasks as completed, with a focus on a seamless user experience.

Role and Contributions:

My role in the project was as a front end developer.
I designed and implemented the user interface and interactive functionality, ensuring a responsive and visually appealing design.

Technologies Used:

HTML
CSS
JavaScript

Features and Functionality:

Add Tasks: Users can add new tasks to their list with a simple input form.
Edit Tasks: Users can edit the details of existing tasks directly within the list.
Delete Tasks:Delete Tasks: Users can remove tasks that are no longer needed, keeping their list organized.
Mark as Completed: Users can mark tasks as completed, visually distinguishing them from pending tasks with a strikethrough or a different color.
Responsive Design: The application is fully responsive, ensuring a smooth user experience on mobile devices, tablets, and desktops.
Data Persistence: Tasks are saved in the browser's local storage, ensuring that the user's task list remains intact even after the browser is closed and reopened.

Learning and Growth:

Enhanced my proficiency in JavaScript for creating dynamic and interactive web applications.
Gained experience in using local storage for data persistence in web applications.
Developed a deeper understanding of responsive web design principles and best practices.

Project title:

Sweet & Savory:

Overview:

Sweet and Savory is a comprehensive food and dessert website designed to showcase a variety of delicious recipes, including their ingredients and preparation methods. The website provides users with an extensive collection of dishes, catering to different tastes and preferences.

Objective:

To create an engaging and visually appealing platform where users can explore and discover new recipes for both savory dishes and desserts, along with detailed ingredient lists and step-by-step instructions.

Role and Contributions:

My role in the project was as a front end developer.
Designed and implemented the user interface, ensuring a seamless and enjoyable user experience. Focused on creating a responsive and aesthetically pleasing design that highlights the diverse range of recipes.

Technologies Used:

HTML
CSS
JavaScript

Features and Functionality:

Recipe Listings: Display a variety of recipes, each with a detailed description, ingredient list, and preparation steps.
Search and Filter: Users can easily search for specific recipes or filter by category (e.g., savory, dessert, vegetarian).
Interactive Recipe Cards: Each recipe is presented on an interactive card that expands to show more details when clicked.
Ingredient Lists: Detailed lists of ingredients for each recipe, including measurements and optional substitutes.
Step-by-Step Instructions: Clear, easy-to-follow instructions for preparing each dish.
Responsive Design: The website is optimized for all screen sizes, ensuring a consistent experience across devices.
Visual Appeal: High-quality images of each dish to entice and inspire users.
